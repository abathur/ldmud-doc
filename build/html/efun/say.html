DESCRIPTION¶

There are two major modes of calling:

If the first argument is a string str, it will be send to all livings in the current room except to the initiator.

If the first argument is an array/mapping/struct/object msg, lfun catch_msg(A) of all living objects except the initiator will be called. This msg will be given as first argument to the lfun, and the initiating object as the second.

Warning

If the lfun catch_msg(A) modifies the content of msg, all subsequent objects will receive the modified msg.

By specifying a second argument to the efun one can exclude more objects than just the initiator. If the second argument is a single object exclude, both the given object and the initiator are excluded from the call. If the second argument is an array excludes, all objects and just the objects in this array are excluded from the call.

The ‘initiator’ is determined according to these rules:

  • if the say(E) is called from within a living object, this becomes the initiator
  • if the say(E) is called from within a dead object as result of a user action (i.e. this_player() is valid), this_player() becomes the initiator.
  • Else the object calling the say(E) becomes the initiator.

USAGE¶

Both of these calls are equal when called by a non-living object:

say("Hi!\n");
say("Hi!\n", this_player());

The next example shows how say(E) works with catch_tell(A). The 2nd object must not be living so the write(E) will go to the current user.

Object 1 (living):

void catch_tell(string str) {
  write("Received: "+str+"\n");
}

Object 2 (not living):

void func() {
  ...
  say("HiHo!\n");
  ...
}

This more complex example demonstrates how say(E) and catch_msg(A) work. Here we also use a non living object to send the message so the who in catch_msg(A) will be the current user.

Object 1 (living):

void catch_msg(mixed *arr, object who) {
  int i;
  if(!arr) return;
  for(i=0; i<sizeof(arr); i++)
    tell_object(who, (stringp(arr[i]) ? arr[i] : "-/-")+"\n");
}

Object 2 (not living):

void func() {
  ...
  say( ({ "Hello", "there!" }) );
  ...
}
