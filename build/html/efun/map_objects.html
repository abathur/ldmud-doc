mixed * map_objects( string *arr, string fun, mixed extra, ...)

DESCRIPTION¶

Similar to map(), but calls arr[n]->fun(extra,...). The returned value replaces the object/string in returned array. <arr> may contain a mixture of objects and object names as well.

Any number of extra arguments is allowed, which are all passed. 0-entries in arr are ignored.
