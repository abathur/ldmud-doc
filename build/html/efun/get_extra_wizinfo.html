mixed get_extra_wizinfo( int wiz)

DESCRIPTION¶

Arguments:
  • wiz (object) – the entry of its creator (uid) is used
  • wiz (string) – a creator (aka uid) that names the entry to use
  • wiz (int) – Uses the given wizlist entry (or the default entry, if the number is 0)

Returns the ‘extra’ information that was set for the given wizard wiz in the wizlist.

The function causes a privilege violation.
