int widthof( mapping map)

DESCRIPTION¶

Returns the number of values per key of mapping map. If map is 0, the result is 0.

USAGE¶

mapping m = ([ "foo": 1,2 ]);
widthof(m) // returns 2

HISTORY¶

  • introduced (3.2.6)
