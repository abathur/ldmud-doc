SYNOPSIS¶

int m_sizeof(mapping map)¶

DESCRIPTION¶

Returns the number of indices in mapping ‘map’. This function is in fact an alias for sizeof().

HISTORY¶

  • changed (3.2.9) – not available if driver is compiled without USE_DEPRECATED.
  • removed (3.3)
