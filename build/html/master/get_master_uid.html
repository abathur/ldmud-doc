SYNOPSIS¶

¶
string get_master_uid( void)

DESCRIPTION¶

Return the string to be used as root-uid. Under !strict_euids, the function is expendable.

HISTORY¶

  • introduced (3.2.1@40) – replaces get_root_uid().
