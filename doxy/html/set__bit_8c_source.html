10 Return the new string where bit n is set in string str. Note
11 that the old string str is not modified.
12 
13 Each character contains 6 bits. So you can store a value
14 between 0 and 63 in one character (2^6=64). Starting character
15 is the blank " " which has the value 0. The first charcter in
16 the string is the one with the lowest bits (0-5).
17 
18 The new string will automatically be extended if needed.
19 
20 EXAMPLES
21 string s;
22 s=set_bit("?",5);
23 
24 Because "?" has a value of 31 the variable s will now contain
25 the character "_" which is equal to 63 (31+2^5=63).
