13 The efun returns 1 if <obj> is a clone, and 0 if it is not.
14 The <obj> can be given as the object itself, or by its name.
15 If <obj> is omitted, the current object is tested.
16 Arguments of other types return 0.
17 Objects with replaced programs no longer count as clones.
18 
19 EXAMPLES
20 object o;
21 o = clone_object("/std/thing");
22 write(clonep(o)); --> writes "1"
23 write(clonep("/std/thing")) --> writes "0"

25 (In COMPAT_MODE use "std/thing" as the filename)
