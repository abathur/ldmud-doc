10 string object_name(object ob)
11 
12 Get the name of an object <ob> or, if no argument is given, of
13 the current object.
14 
15 As a special case, if <ob> is 0, the function returns 0.
16 
17 This name is the name under which the object is stored in the
18 muds object table. It is initialised at the creation of the
19 object such that blueprints are named after the file they are
20 compiled from (without the trailing '.c'), and clones receive
21 the name of their blueprint, extended by '#' followed by
22 a unique non-negative number. These rules also apply to
23 virtual objects - the real name/type of virtual objects
24 is ignored.
25 
26 The name of an object can be changed with rename_object(), and
27 object_name() will reflect any of these changes.
28 
29 The returned name always begins with '/' (absolute path),
30 except when the parser runs in COMPAT mode.
31 
32 EXAMPLES
33  find_object(object_name(ob)) == ob
34 
35 This is guaranteed to be true for all objects ob that are not
36 destructed.
37 
38 0 is accepted as argument since 3.2.9.
