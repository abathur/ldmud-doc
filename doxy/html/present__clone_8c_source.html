3  * This efun searches the inventory of object <env> (default is
4  * this_object()) for an object with a specific blueprint.
5  * The blueprint can be specified either by name <str>, or as the same
6  * blueprint as of object <obj>. The matching criteria in both cases is
7  * the load_name().
8  * If <n> is given, the <n>th object in <env> is returned (if present),
9  * otherwise the first object matching <str> or <obj>.
10  * If no object in <env> matches the criteria, 0 is returned.
11  *
12  * For plain driver this name starts with a '/', for COMPAT mode
13  * drivers it doesn't.
14  *
15  * EXAMPLES
16  * Assume that object 'env' contains the objects /obj/money#8,
17  * /std/weapon#9, /std/weapon#12 and /obj/key in the given order.
18  *
19  * +----------------------------------------------------+-----------------+
20  * | Function call | returns |
21  * +----------------------------------------------------+-----------------+
22  * | present_clone("/obj/money", env) | /obj/money#8 |
23  * | present_clone("/std/weapon#12", env) | /std/weapon#9 |
24  * | present_clone(find_object("/obj/money#14"), env) | /obj/money#8 |
25  * | present_clone("/obj/key#18", env) | /obj/key |
26  * | present_clone("/std/weapon#12", env, 2) | /std/weapon#12 |
27  * | present_clone("/std/weapon#12", env, 3) | 0 |
28  * +----------------------------------------------------+-----------------+
29  *
30  * REMARKS
31  * Note that in contrast to present(), this efun never searches
32  * in the environment of <env>.
33  *
34  * Introduced in 3.2.7.
35  * Searching for the <n>th object was added in 3.3.718.
36  *
37  * @synopsis{
38  * object present_clone(string str)
39  * object present_clone(string str, int n)
40  * object present_clone(string str, object env)
41  * object present_clone(string str, object env, int n)
42  * object present_clone(object obj)
43  * object present_clone(object obj, int n)
44  * object present_clone(object obj, object env)
45  * object present_clone(object obj, object env, int n)
46  * }
47  *
48  * @see efun::load_name(), efun::present()
49  **/
50  object present_clone(object obj, object env, int n);
