2  /**
3  * This is a privileged function, only to be used in the master
4  * object or in the simul_efun object.
5  *
6  * It changes the result of this_object() in the using function,
7  * and the result of previous_object() in functions called in
8  * other objects by call_other()/apply()/funcall(). Its effect will
9  * remain till there is a return of an external function call, or
10  * another call of set_this_object(). While executing code in the
11  * master object's program or the primary simul_efun object's
12  * program, set_this_object() is granted even if this_object() is
13  * altered by set_this_object(). This does not apply to functions
14  * inherited from other programs.
15  *
16  * Use it with extreme care to avoid inconsistencies. After a
17  * call of set_this_object(), some LPC-constructs might behave in
18  * an odd manner, or even crash the system. In particular, using
19  * global variables or calling local functions (except by
20  * call_other) is illegal and actively prevented.
21  *
22  * Allowed are call_other(), map functions, access of local variables
23  * (which might hold array pointers to a global array), simple arithmetic
24  * and the assignment operators.
25  *
26  * @synopsis{
27  * void set_this_object(object object_to_pretend_to_be)
28  * }
29  *
30  **/
31  void set_this_object(object object_to_pretend_to_be);
