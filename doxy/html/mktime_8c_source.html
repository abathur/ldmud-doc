12 If the argument <ts> is an array with 9 elements (int) according to
13 the result of local_time()/gm_time(), this function returns the number
14 of seconds passed since the epoch (00:00:00 UTC, January 1, 1970).
15 This can be used to store a date or time as an integer value or to
16 compute differences betweens two different dates or times.
17 
18 The array <ts> has to have the following structure:
19  int TM_SEC (0): seconds (0..59)
20  int TM_MIN (1): minutes (0..59)
21  int TM_HOUR (2): hours (0..23)
22  int TM_MDAY (3): day of month (1..31)
23  int TM_MON (4): day of year (0..11)
24  int TM_YEAR (5): year (e.g. 2001)
25  int TM_WDAY (6): day of week (0..6, sunday = 0)
26  int TM_YDAY (7): day of year (0..365)
27  inz TM_ISDST (8): TRUE: summer time is in effect.
28 
29 TM_YDAY and TM_WDAY are ignored and can contain arbitrary
30 integer values.
31 
32 EXAMPLES
33 A date and time (user input) shall be stored as unix timestamp:
34 // "Wed Oct 24 10:48:00 2007" corresponds to the returned time stamp:
35 int unixtime = mktime( ({0, 48, 09, 24, 09, 2007, 0, 01, 0}) );
