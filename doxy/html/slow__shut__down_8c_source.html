2  /**
3  * Schedule a shutdown for the near future. minutes is the
4  * desired time in minutes till the shutdown:
5  *
6  * - six, if just the user reserve has been put into use.
7  * - one, if the (smaller) master reserve has been put into use as
8  * well.
9  *
10  * The interpreter calls this function when it runs low on
11  * memory. At this time, it has freed its reserve, but since it
12  * won't last long, the interpreter needs to be shut down. The
13  * delay is to give the users the opportunity to finish their
14  * current tasks, but don't take the 'minutes' for granted, just
15  * deduce the urgency from it.
16  *
17  * It is possible that the driver may reallocate some memory
18  * after the function has been called, and then run again into a
19  * low memory situation, calling this function again.
20  *
21  * This function might load an 'Armageddon' object and tell it
22  * what to do. It is the Armageddon object then which performs
23  * the shutdown.
24  *
25  * Technical: The memory handling of the interpreter includes
26  * three reserved areas: user, system and master. All three are
27  * there to insure that the system shuts down gracefully when the
28  * memory runs out: the user area to give the users time to
29  * quit normally, the others to enable emergency-logouts when the
30  * user reserve is used up as well.
31  *
32  * The areas are allocated at start of the interpreter, and
33  * released when no more memory could be obtained from the host.
34  * In such a case, one of the remaining areas is freed (so the
35  * operation can continue a short while) and a garbage collection
36  * is initiated. If the garbage collection recycles enough memory
37  * (either true garbage or by the aid of the quota_demon) to
38  * reallocate the areas, all is fine, else the system shut down
39  * is invoked by a call to this function.
40  *
41  * @synopsis{
42  * void slow_shut_down(int minutes)
43  * }
44  *
45  * @see master::quota_demon(), master::notify_shutdown(), efun::shutdown(), @ref driver_concepts_memory "memory", master::low_memory()
46  **/
47  void slow_shut_down(int minutes);
48 }
