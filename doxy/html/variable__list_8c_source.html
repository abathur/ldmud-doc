2  /**
3  * Return an array with information about \p ob variables. For every variable, 1 to 3 values (depending on \p flags) are stored in the result array conveying in this order:
4  *
5  * - the name of the variable
6  * - the variable flags (see below)
7  * - the return type (listed in <lpctypes.h>)
8  * - the value of the variable
9  *
10  * \p ob may be given as true object or as a filename. In the latter case, the efun does not try to load the object before proceeding.
11  *
12  * If \p ob is not the current object and the value of the variable is requested, a privilege_violation ("variable_list", <ob>) occurs.
13  *
14  * \p flags determines both which information is returned for every variable, and which variables should be considered at all. Its value is created by bin-or'ing together the following flags from <functionlist.h>:
15  *
16  * Control of returned information:
17  *
18  * `RETURN_FUNCTION_NAME` include the variable name
19  * `RETURN_FUNCTION_FLAGS` include the variable flags
20  * `RETURN_FUNCTION_TYPE` include the return type
21  * `RETURN_VARIABLE_VALUE` include the variable value
22  *
23  * Control of listed variables:
24  *
25  * `NAME_INHERITED` don't list if defined by inheritance
26  * `TYPE_MOD_STATIC` ==
27  * `TYPE_MOD_NOSAVE` don't list if nosave ('static') variable
28  * `TYPE_MOD_PRIVATE` don't list if private
29  * `TYPE_MOD_PROTECTED` don't list if protected
30  * `NAME_HIDDEN` don't list if not visible through inheritance
31  *
32  * The 'flags' information consists of the bin-or of the list control flags given above, plus the following:
33  *
34  * `TYPE_MOD_VIRTUAL` variable is inherited virtually
35  * `TYPE_MOD_NO_MASK` variable is nomask
36  * `TYPE_MOD_PUBLIC` variable is public
37  *
38  * All these flags are defined in <functionlist.h>; the return types are defined in <lpctypes.h>.
39  *
40  * @headerfile <lpctypes.h>
41  *
42  * @headerfile <functionlist.h>
43  *
44  * @synopsis{
45  * mixed * variable_list(object ob, int flags)
46  * }
47  *
48  * @history{
49  * introduced (3.2.10)
50  * }
51  *
52  * @see efun::inherit_list(), efun::functionlist(), efun::variable_exists()
53  **/
54  mixed * variable_list(object ob, int flags);
55 }
