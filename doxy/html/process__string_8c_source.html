10 Searches string str for occurences of a "value by function
11 call", which is an implicit function call surrounded by @@. See
12 "value_by_function_call" in the principles section.
13 
14 The value should contain a string like this:
15 @@function[:filename][>>|<<arg|arg]@@
16 
17 <function> must return a string or else the string which should be
18 processed will be returned unchanged.
19 
20 process_string() does not recurse over returned
21 replacement values. If a function returns another function
22 description, that description will not be replaced.
23 
24 Both the filename and the args are optional.
25 
26 Consecutive function calls can be written adjacent:
27 
28  @@function1@@function2@@
29 
30 EXAMPLES
31 string foo(string str) {
32 
33  return "ab"+str+"ef";
34 
35 }
36 void func() {
37 
38  write(process_string("@@foo|cd@@")+"n");
39 
40 }
41 
42 The function func() will print out the string "abcdef".
43 
44 BUGS
45 Using process_string() can lead to severe security problems.
46
