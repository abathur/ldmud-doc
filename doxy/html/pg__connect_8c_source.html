6  *
7  * The \p conn string is in the format accepted by Postgres' `PQconnectStart()` API functions. Pass an empty string to use the default options, or a string holding the `'<key>=<value>'` options separated by whitespace.
8  *
9  * The most useful options are:
10  *
11  * dbname
12  * The database name
13  *
14  * user
15  * The user name to connect as.
16  *
17  * password
18  * Password to be used.
19  *
20  * Return 0 on success, and -1 on failure.
21  *
22  * The function is available only if the driver is compiled with PostgreSQL support. In that case, @ref driver_topics_predefined "__PGSQL__" is defined.
23  *
24  * The efun triggers a privilege violation ("pgsql", "pg_connect").
