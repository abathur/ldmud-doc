10 Copy the bitrange [<srcstart>..<srcstart>+<copylen>[ from
11 bitstring <src> and copy it into the bitstring <dest> starting
12 at <deststart>, overwriting the original bits at those positions.
13 
14 The resulting combined string is returned, the input strings remain
15 unaffected.
16 
17 If <srcstart> is not given, <src> is copied from the start.
18 If <srcstart> is negative, it is counted from one past the last set
19 bit in the string (ie. '-1' will index the last set bit).
20 
21 If <deststart> is not given, <dest> will be overwritten from the start.
22 If <deststart> is negative, it is counted from one past the last set
23 bit in the string (ie. '-1' will index the last set bit).
24 
25 If <copylen> is not given, it is assumed to be infinite, ie. the result
26 will consist of <dest> up to position <deststart>, followed by
27 the data copied from <src>.
28 If <copylen> is negative, the function will copy the abs(<copylen>)
29 bits _before_ <srcstart> in to the result.
30 
31 None of the range limits can become negative.
32 
33 EXAMPLES
34 copy_bits(src, dest, 10) === src[10..]
35 copy_bits(src, dest, 10, 5) === dest[0..4] + src[10..]
36 copy_bits(src, dest, 10, 5, 3)
37 
38  === dest[0..4] + src[10..12] + dest[8..]
39 
40  (The src[]/dest[] is just for explanatory purposes!)
