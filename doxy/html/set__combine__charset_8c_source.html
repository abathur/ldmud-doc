12 Set the set of characters which can be combined into a single string
13 when already received en-bloc in charmode from the current interactive
14 user.
15 
16 Non-combinable characters and single received characters are returned
17 as separate strings as usual. The function must be called with the
18 interactive user being the command giver.
19 
20 The newline 'n' and the NUL character '0' are always non-combinable.
21 
22 The charset can be given either directly as a string, or indirectly
23 as a bitvector. If given as the number 0, the default combine
24 charset is re-established.
25 
26 The bitvector is interpreted as an array of 8-bit-values and might
27 contain up to 32 elements. Character n is allowed to be output
28 if sizeof(bitvector) > n/8 && bitvector[n/8] & (1 << n%8) .
